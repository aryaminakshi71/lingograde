# Shared Templates

These are simplified template pages used in this SAAS application.

## Contents
- `app/` - Next.js App Router template pages
  - `page.tsx` - Landing page template
  - `dashboard/page.tsx` - Dashboard template
  - `settings/page.tsx` - Settings page template
  - `pricing/page.tsx` - Pricing page template

## Usage
These templates are used as base templates for the application pages.

## Note
These are simplified demonstration templates, not full-featured pages.
